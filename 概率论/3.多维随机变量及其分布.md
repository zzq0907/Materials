## 3.1 多维随机变量及其联合分布

> **核心概念**：实际问题往往需要同时用多个随机变量来描述，多维随机变量及其联合分布描述了它们之间的统计规律。

### 1. 多维随机变量

设 $X_1, X_2, \ldots, X_n$ 是定义在同一概率空间 $(\Omega, \mathcal{F}, P)$ 上的随机变量，则称

$$(X_1, X_2, \ldots, X_n)$$

为 $n$ 维随机变量（随机向量）。

> **例**：掷骰子两次，$(X_1, X_2)$ 为 2 维随机向量；身高和体重构成 2 维随机向量。
>
> 以下主要讨论 $n=2$ 的情形。

### 2. 联合分布函数

**定义**：对任意 $n$ 个实数 $x_1, x_2, \ldots, x_n$，$n$ 元函数

$$F(x_1, x_2, \ldots, x_n) = P(X_1 \leq x_1, X_2 \leq x_2, \ldots, X_n \leq x_n)$$

称为 $n$ 维随机变量的**联合分布函数**。

**二维分布函数的性质**：
1. **单调性**：对每个变元单调不减
2. **有界性**：$0 \leq F(x,y) \leq 1$，且 $F(-\infty, y) = F(x, -\infty) = 0$，$F(+\infty, +\infty) = 1$
3. **右连续性**：对每个变元右连续
4. **非负性（相容性）**：对任意 $a < b, c < d$，
   $$F(b,d) - F(b,c) - F(a,d) + F(a,c) \geq 0$$

> **注**：性质（4）是二维分布函数特有的，不能由前三条推出。满足以上四条性质的二元函数一定是某个二维随机变量的分布函数。

### 3. 联合分布列（离散型）

若二维 r.v. $(X, Y)$ 的所有可能取值为有限或可列多个，则称为**二维离散型 r.v.**。

设 $(X, Y)$ 的所有可能取值为 $(x_i, y_j)$，则称

$$p_{ij} = P(X = x_i, Y = y_j), \quad i, j = 1, 2, \ldots$$

为 $(X, Y)$ 的**联合分布列**。

**性质**：
1. $p_{ij} \geq 0$
2. $\sum_i \sum_j p_{ij} = 1$

**表示形式**：常用表格表示联合分布列。

### 4. 联合密度函数（连续型）

若存在非负可积函数 $p(x, y)$，使得对任意 $x, y$：

$$F(x, y) = \int_{-\infty}^{x} \int_{-\infty}^{y} p(u, v) \, du \, dv$$

则称 $(X, Y)$ 为**二维连续型 r.v.**，$p(x, y)$ 为**联合密度函数**。

**性质**：
1. $p(x, y) \geq 0$
2. $\int_{-\infty}^{+\infty} \int_{-\infty}^{+\infty} p(x, y) \, dx \, dy = 1$
3. $P((X, Y) \in D) = \iint_D p(x, y) \, dx \, dy$

## 3.2 边际分布与随机变量的独立性

> **核心概念**：边际分布从联合分布中"投影"出单个变量的分布；独立性使联合分布由边际分布唯一确定。

### 1. 边际分布函数

由联合分布函数 $F(x, y)$ 可得：

**$X$ 的边际分布函数**：$F_X(x) = F(x, +\infty) = \lim_{y \to +\infty} F(x, y)$

**$Y$ 的边际分布函数**：$F_Y(y) = F(+\infty, y) = \lim_{x \to +\infty} F(x, y)$

> **联合 → 边际**：联合分布函数确定，边际分布函数完全确定。
>
> **边际 → 联合**：一般不能！联合分布还包含各分量间的关系信息。

### 2. 边际分布列（离散型）

$$P(X = x_i) = \sum_j p_{ij} \triangleq p_{i \bullet}, \quad P(Y = y_j) = \sum_i p_{ij} \triangleq p_{\bullet j}$$

### 3. 边际密度函数（连续型）

$$p_X(x) = \int_{-\infty}^{+\infty} p(x, y) \, dy, \quad p_Y(y) = \int_{-\infty}^{+\infty} p(x, y) \, dx$$

### 4. 随机变量的相互独立性

设 $(X, Y)$ 为二维 r.v.，若对任意实数 $x, y$，有

$$F(x, y) = F_X(x) \cdot F_Y(y)$$

则称 $X$ 与 $Y$ **相互独立**。

**等价条件**：
- **离散型**：$p_{ij} = p_{i \bullet} \cdot p_{\bullet j}$（对所有 $i, j$）
- **连续型**：$p(x, y) = p_X(x) \cdot p_Y(y)$（几乎处处成立）

> **独立性的意义**：当 $X$ 与 $Y$ 独立时，**边际分布完全确定联合分布**！

**重要命题**：若 $X$ 与 $Y$ 相互独立，则 $U = u(X)$ 与 $V = v(Y)$ 也相互独立（独立 r.v. 的函数仍独立）。

**判别命题**（连续型）：若 $p(x, y) = r(x) \cdot g(y)$（可分离变量形式），则 $X$ 与 $Y$ 相互独立，且边际密度分别正比于 $r(x)$ 和 $g(y)$。

> **注意**：两随机变量独立且有相同分布，不能推出它们相等！例如独立同分布的 $X$ 和 $Y$，$P(X=Y) \neq 1$。

## 3.3 二维随机变量函数的分布

> **核心方法**：已知 $(X, Y)$ 的分布，求 $Z = g(X, Y)$ 的分布。连续情形从分布函数出发是通用方法。

### 离散情形

当 $(X, Y)$ 为离散 r.v. 时，$Z = g(X, Y)$ 也是离散 r.v.，直接枚举所有 $(x_i, y_j)$ 计算 $Z$ 的取值及对应概率。

**可加性**（独立随机变量之和）：
- $X \sim b(n_1, p), Y \sim b(n_2, p)$ 且独立 $\Rightarrow$ $X + Y \sim b(n_1 + n_2, p)$
- $X \sim P(\lambda_1), Y \sim P(\lambda_2)$ 且独立 $\Rightarrow$ $X + Y \sim P(\lambda_1 + \lambda_2)$

### 连续情形：$Z = X + Y$ 的分布（卷积）

设 $(X, Y)$ 的联合密度为 $p(x, y)$，则 $Z = X + Y$ 的密度为：

$$p_Z(z) = \int_{-\infty}^{+\infty} p(x, z - x) \, dx = \int_{-\infty}^{+\infty} p(z - y, y) \, dy$$

当 $X$ 与 $Y$ 独立时（**卷积公式**）：

$$p_Z(z) = \int_{-\infty}^{+\infty} p_X(x) \cdot p_Y(z - x) \, dx = \int_{-\infty}^{+\infty} p_X(z - y) \cdot p_Y(y) \, dy$$

> **正态分布的可加性**：若 $X_i \sim N(\mu_i, \sigma_i^2)$ 且相互独立，则
> $$\sum_i a_i X_i \sim N\left(\sum_i a_i \mu_i,\; \sum_i a_i^2 \sigma_i^2\right)$$

### 变量代换法（增补变量法）

求 $Z = g(X, Y)$ 的分布的另一方法：
1. 引入辅助变量 $V$（通常 $V = X$ 或 $V = Y$）
2. 求 $(Z, V)$ 的联合密度（利用雅可比行列式）
3. 求 $Z$ 的边际密度

设变换 $\begin{cases} z = g(x,y) \\ v = h(x,y) \end{cases}$ 有唯一反函数 $\begin{cases} x = x(z,v) \\ y = y(z,v) \end{cases}$，则

$$p_{ZV}(z, v) = p_{XY}(x(z,v), y(z,v)) \cdot \left| \frac{\partial(x, y)}{\partial(z, v)} \right|$$

其中 $J = \frac{\partial(x, y)}{\partial(z, v)} = \begin{vmatrix} \frac{\partial x}{\partial z} & \frac{\partial x}{\partial v} \\ \frac{\partial y}{\partial z} & \frac{\partial y}{\partial v} \end{vmatrix}$ 为雅可比行列式。

## 3.4 多维随机变量的数字特征

> **核心概念**：协方差和相关系数刻画了两个随机变量之间的线性相关程度，是多元统计的基础。

### 1. 多维随机变量函数的数学期望

设 $Z = f(X, Y)$，则：

- 离散：$E(Z) = \sum_i \sum_j f(x_i, y_j) p_{ij}$
- 连续：$E(Z) = \int_{-\infty}^{+\infty} \int_{-\infty}^{+\infty} f(x, y) p(x, y) \, dx \, dy$

### 2. 期望与方差的性质

- $E(aX + bY) = aE(X) + bE(Y)$（可推广到有限个）
- 若 $X$ 与 $Y$ 独立 $\Rightarrow$ $E(XY) = E(X)E(Y)$
- 若 $X$ 与 $Y$ 独立 $\Rightarrow$ $\text{Var}(X \pm Y) = \text{Var}(X) + \text{Var}(Y)$

### 3. 协方差

**定义**：

$$\text{Cov}(X, Y) = E[(X - E(X))(Y - E(Y))]$$

**计算公式**：

$$\text{Cov}(X, Y) = E(XY) - E(X)E(Y)$$

**任意两变量的方差和公式**：

$$\text{Var}(X \pm Y) = \text{Var}(X) + \text{Var}(Y) \pm 2\,\text{Cov}(X, Y)$$

**协方差的性质**：
1. $\text{Cov}(X, X) = \text{Var}(X)$
2. $\text{Cov}(X, Y) = \text{Cov}(Y, X)$
3. $\text{Cov}(aX, bY) = ab \,\text{Cov}(X, Y)$
4. $\text{Cov}(X_1 + X_2, Y) = \text{Cov}(X_1, Y) + \text{Cov}(X_2, Y)$
5. 若 $X$ 与 $Y$ 独立，则 $\text{Cov}(X, Y) = 0$。**但反之不成立！**

> **不相关 $\not\Rightarrow$ 独立**：协方差（或相关系数）为零只表示没有**线性**关系，但可能存在非线性关系。

### 4. 相关系数

**定义**：

$$\rho_{XY} = \text{Corr}(X, Y) = \frac{\text{Cov}(X, Y)}{\sqrt{\text{Var}(X) \cdot \text{Var}(Y)}}$$

> 相关系数是对标准化后的变量求协方差，消除了量纲的影响。

**性质**（Schwarz 不等式保证）：
1. $|\rho_{XY}| \leq 1$
2. $|\rho_{XY}| = 1$ 的充要条件是存在常数 $a, b$（$a \neq 0$），使得 $P(Y = a + bX) = 1$
3. 若 $X$ 与 $Y$ 独立，则 $\rho_{XY} = 0$；但反之不成立

**相关系数的含义**：
- $\rho > 0$：正相关（$X$ 增大时 $Y$ 也倾向于增大）
- $\rho < 0$：负相关（$X$ 增大时 $Y$ 倾向于减小）
- $\rho = 0$：不相关（无线性关系）
- $\rho = \pm 1$：完全线性相关

> **最优线性预测**：以 $a + bX$ 近似 $Y$ 的均方误差最小时，$b = \rho_{XY} \frac{\sqrt{\text{Var}(Y)}}{\sqrt{\text{Var}(X)}}$。相关系数越大，线性近似越好。

## 3.5 条件分布和条件期望

> **核心概念**：条件分布在给定一个变量取值时，描述了另一个变量的分布规律。条件期望是条件分布的数学期望。

### 1. 条件分布列（离散型）

在 $X = x_i$ 的条件下，$Y$ 的**条件分布列**为：

$$P(Y = y_j \mid X = x_i) = \frac{P(X = x_i, Y = y_j)}{P(X = x_i)} = \frac{p_{ij}}{p_{i \bullet}}$$

> 类似乘法公式：$p_{ij} = P(Y = y_j \mid X = x_i) \cdot P(X = x_i)$
>
> 类似全概率公式：$P(Y = y_j) = \sum_i P(Y = y_j \mid X = x_i) \cdot P(X = x_i)$

### 2. 条件密度函数（连续型）

在 $Y = y$ 的条件下，$X$ 的**条件密度函数**为：

$$p_{X \mid Y}(x \mid y) = \frac{p(x, y)}{p_Y(y)}, \quad p_Y(y) > 0$$

类似地：

$$p_{Y \mid X}(y \mid x) = \frac{p(x, y)}{p_X(x)}, \quad p_X(x) > 0$$

> 类似乘法公式：$p(x, y) = p_{X \mid Y}(x \mid y) \cdot p_Y(y) = p_{Y \mid X}(y \mid x) \cdot p_X(x)$
>
> 类似全概率公式：$p_X(x) = \int_{-\infty}^{+\infty} p_{X \mid Y}(x \mid y) \cdot p_Y(y) \, dy$
>
> 类似 Bayes 公式：$p_{Y \mid X}(y \mid x) = \frac{p_{X \mid Y}(x \mid y) \cdot p_Y(y)}{\int p_{X \mid Y}(x \mid t) \cdot p_Y(t) \, dt}$

### 3. 条件数学期望

**离散情形**：

$$E(Y \mid X = x_i) = \sum_j y_j \cdot P(Y = y_j \mid X = x_i)$$

**连续情形**：

$$E(Y \mid X = x) = \int_{-\infty}^{+\infty} y \cdot p_{Y \mid X}(y \mid x) \, dy$$

### 4. 重期望公式

设 $(X, Y)$ 为二维 r.v.，且 $E(Y)$ 存在，则：

$$E(Y) = E[E(Y \mid X)]$$

即：**无条件期望 = 条件期望的期望**。

> 离散情形：$E(Y) = \sum_i E(Y \mid X = x_i) \cdot P(X = x_i)$
>
> 连续情形：$E(Y) = \int_{-\infty}^{+\infty} E(Y \mid X = x) \cdot p_X(x) \, dx$

## 本章脉络总结

### 知识递进关系

```
多维随机变量 (X₁, X₂, ..., Xₙ)
        ↓
联合分布函数 F(x₁, ..., xₙ)
   ↙          ↘
离散型           连续型
联合分布列 pᵢⱼ    联合密度 p(x,y)
        ↓
边际分布（X 或 Y 单独的分布）
        ↓
独立性判断（F(x,y) = F_X(x)·F_Y(y)）
        ↓
   ┌────────────────────────────────┐
   ↓                                ↓
函数的分布                        数字特征
├─ Z = X+Y（卷积公式）            ├─ 协方差 Cov(X,Y)
├─ Z = X/Y, XY 等                ├─ 相关系数 ρ_XY
├─ 变量代换法                    └─ 方差和公式
└─ 可加性                          
        ↓
条件分布与条件期望
├─ 条件分布列/条件密度
├─ 条件期望 E(Y|X)
└─ 重期望公式 E(Y) = E[E(Y|X)]
```

### 核心对比表

| 概念 | 离散型 | 连续型 |
|------|--------|--------|
| 联合分布 | $p_{ij} = P(X=x_i, Y=y_j)$ | $p(x,y)$ |
| 边际分布 | $p_{i\bullet} = \sum_j p_{ij}$ | $p_X(x) = \int p(x,y)dy$ |
| 独立性 | $p_{ij} = p_{i\bullet} \cdot p_{\bullet j}$ | $p(x,y) = p_X(x) \cdot p_Y(y)$ |
| 条件分布 | $P(Y=y_j \mid X=x_i) = p_{ij}/p_{i\bullet}$ | $p_{Y\mid X}(y\mid x) = p(x,y)/p_X(x)$ |
| 条件期望 | $\sum_j y_j P(Y=y_j \mid X=x_i)$ | $\int y \, p_{Y\mid X}(y\mid x) dy$ |

### 独立与不相关的关系

| 条件 | 结论 |
|------|------|
| $X, Y$ 独立 | $\Rightarrow$ 不相关（$\rho_{XY} = 0$） |
| $X, Y$ 不相关（$\rho_{XY} = 0$） | $\not\Rightarrow$ 独立 |
| $(X, Y)$ 服从二维正态 | 独立 $\Leftrightarrow$ 不相关 |

### 考试应对策略

1. **联合分布列**：熟练使用表格表示和边际求和
2. **独立性验证**：检查 $F(x,y) = F_X(x)F_Y(y)$ 或密度/分布列的可分离性
3. **卷积公式**：$Z = X + Y$ 的分布是计算重点，注意积分区间
4. **协方差与相关系数**：用 $E(XY) - E(X)E(Y)$ 计算最简便
5. **不相关 ≠ 独立**：这是考试常考的陷阱
6. **条件期望**：重期望公式 $E(Y) = E[E(Y \mid X)]$ 是解题利器
