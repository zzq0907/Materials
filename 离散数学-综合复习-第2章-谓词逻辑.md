## 2.1 基本概念 🔴

### 为什么要学谓词逻辑

命题逻辑只能处理"整句"之间的关系（如"如果下雨，那么地湿"），但无法处理句子**内部**的结构。苏格拉底三段论：
> 凡是人都要死的。苏格拉底是人。所以苏格拉底是要死的。

在命题逻辑中只能符号化为 (p∧q)→r，这**并非重言式**！因为 p、q、r 之间没有结构联系。谓词逻辑将句子拆成"对象 + 性质/关系"，从而捕捉内部逻辑结构。

### 个体词

- **个体词**：研究对象，类似于句子中的"主语"。
  - **个体常元**：具体、确定的对象。用小写字母 a, b, c 表示。如"苏格拉底"。
  - **个体变元**：不确定的、泛指的对象。用 x, y, z 表示。如"某个人"。
- **个体域**：个体变项的取值范围
  - 有限个体域：如 {a,b,c}, {1,2}
  - 无限个体域：如 N, Z, R
  - **全总个体域**：宇宙间一切事物

### 谓词

- **谓词**：描述个体性质或个体间关系的词。用 F, G, H 表示。
  - **谓词常项**：表示具体性质。如 F(a)：a 是人。
  - **谓词变项**：表示抽象/泛指。如 F(x)：x 具有性质 F。
- 按元数分类：
  - **一元谓词**：描述个体性质。如 P(x)：x 是质数。
  - **二元谓词**：描述两个个体之间的关系。如 L(x,y)：x 大于 y。
  - **0元谓词**：不含个体变项，退化为命题。如 F(a)。

> **命题逻辑 vs 谓词逻辑**：命题逻辑里"苏格拉底是人"是一个原子命题 p。谓词逻辑里它拆成 P(a)，其中 P 是谓词"…是人"，a 是个体常元"苏格拉底"。

### 量词

- **全称量词 ∀**："所有的"、"每一个"。∀xP(x) 读作"对所有 x，P(x) 成立"。
- **存在量词 ∃**："存在"、"至少有一个"。∃xP(x) 读作"存在 x，使得 P(x) 成立"。

> **∀ 和 ∃ 的关系**：它们通过否定互相转化。¬∀xP(x) ⇔ ∃x¬P(x)，¬∃xP(x) ⇔ ∀x¬P(x)。

### 命题符号化的核心模式

**限定个体域时**：
- "所有 x 都具有性质 G"：∀x G(x)
- "存在 x 具有性质 G"：∃x G(x)

**使用全总个体域时**（需引入**特性谓词**）：
- "所有 F 都是 G"：∀x(F(x)→G(x))
- "存在 F 是 G"：∃x(F(x)∧G(x))

### PPT经典例题

**例**（不同个体域下符号化）：
(1) 人都爱美；(2) 有人用左手写字

- 个体域为人类集合：(1) ∀x G(x)；(2) ∃x G(x)
- 个体域为全总个体域：(1) ∀x(F(x)→G(x))；(2) ∃x(F(x)∧G(x))

其中 F(x) 为特性谓词，表示"x 是人"。

**例**（复杂命题符号化）：
(1) 正数都大于负数。
- 解：令 F(x): x为正数, G(y): y为负数, L(x,y): x>y
- ∀x(F(x)→∀y(G(y)→L(x,y))) 或 ∀x∀y((F(x)∧G(y))→L(x,y))

(2) 有的无理数大于有的有理数。
- ∃x∃y(F(x)∧G(y)∧L(x,y))

(3) 没有不吃饭的人。
- ¬∃x(F(x)∧¬G(x)) ⇔ ∀x(F(x)→G(x))

(4) 素数不全是奇数。
- ¬∀x(F(x)→G(x)) ⇔ ∃x(F(x)∧¬G(x))

### 符号化注意事项

1. 未指定个体域时默认使用全总个体域
2. 使用全总个体域时须引入特性谓词
3. 表达多个个体词关系时使用多元谓词
4. 同一命题可能存在多种符号化方式
5. **不同量词间的顺序一般不能随便颠倒**
   - 如 ∀x∃yL(x,y) 与 ∃y∀xL(x,y) 含义不同

## 2.2 一阶逻辑公式 🔴

### 指导变元、辖域、约束出现、自由出现

以公式 ∀x(P(x)→∃y Q(x,y)) ∨ R(x) 为例：

- **指导变元**：量词后面紧跟的那个变元。∀ 的指导变元是 x，∃ 的指导变元是 y。
- **辖域**：量词的作用范围。∀x 的辖域是 P(x)→∃y Q(x,y)。∃y 的辖域是 Q(x,y)。
- **约束出现**：在量词辖域内、与指导变元同名的变元出现。
- **自由出现**：不在任何量词辖域内的变元出现。最后一个 R(x) 中的 x 是自由出现。

> **一句话判断**：变元前面有"管着它"的量词就是约束的，没有就是自由的。

### 一阶逻辑的字母表

- 个体常项符号：a,b,c,...
- 个体变项符号：x,y,z,...
- 函数符号：f,g,h,...
- 谓词符号：F,G,H,...
- 量词符号：∀, ∃
- 联结词符号：¬, ∧, ∨, →, ↔
- 括号与逗号

### 项 (Term)

1. 个体常项和个体变项是项
2. 若 f 是 n 元函数符号，t₁,...,tₙ 是项，则 f(t₁,...,tₙ) 是项
3. 只有有限次应用上述规则得到的是项

### 原子公式与合式公式

- **原子公式**：F(t₁, t₂, ..., tₙ)，其中 F 是 n 元谓词，t₁...tₙ 是项。
- **合式公式**：递归定义（在命题逻辑的基础上增加量词规则）。

### 闭式

- **闭式**：不含自由变元的公式。闭式的真值是确定的（在给定解释下）。
- **换名规则**：将量词中约束变元换名，保证不同量词约束不同变元，且不与自由变元同名。

> **为什么重要**：有自由变元的公式真值不固定（因为自由变元没被量化），只有闭式才像一个"完整的命题"。

## 2.3 解释与赋值 🔴

### 给公式赋予含义

命题逻辑里，一个公式的真值只需要给每个命题变元一个 0/1。谓词逻辑更复杂——需要指定"讨论的对象是谁"、"谓词什么意思"。

- **解释 I**：由四部分组成：
  1. **个体域 D**：讨论范围
  2. **常元解释**：每个常元对应 D 中的一个元素
  3. **函数解释**：每个函数符号对应 D 上的一个函数
  4. **谓词解释**：每个谓词符号对应 D 上的一个关系

- **赋值 σ**：给自由变元指定个体域中的具体值。（约束变元不需要赋值，它们归量词管。）

> **通俗理解**：解释 = "我们讨论的是什么世界"（自然数？人类？）。赋值 = "自由变元具体取什么值"。两者合起来才能判断一个公式在这个世界里的真值。

### 谓词逻辑中的公式类型

| 类型 | 定义 | 与命题逻辑的对应 |
|------|------|------------------|
| **逻辑有效式（永真式）** | 在任何解释和赋值下都为真 | 重言式 |
| **矛盾式（永假式）** | 在任何解释和赋值下都为假 | 矛盾式 |
| **可满足式** | 至少存在一个解释和赋值使其为真 | 可满足式 |

> **重要区别**：命题逻辑里真值表判永真只需 2ⁿ 种赋值。谓词逻辑里个体域可以是无穷集，没法穷举。

### 代换实例

将命题逻辑中的重言式中的命题变项替换为一阶逻辑公式，得到的仍为永真式。

例如：p→(p∨q) 是重言式，则 F(x)→(F(x)∨G(y)) 是永真式。

## 2.4 等值式与前束范式 🔴

### 重要等值式

**量词否定等值式**（最核心，必背）：
- ¬∀xA(x) ⇔ ∃x¬A(x)（"并非所有都是"等价于"至少有一个不是"）
- ¬∃xA(x) ⇔ ∀x¬A(x)（"不存在"等价于"所有都不是"）

> **记忆口诀**："∀变∃，∃变∀，否定往右推"

**量词分配等值式**：
- ∀x(A(x)∧B(x)) ⇔ ∀xA(x)∧∀xB(x)  （∀ 对 ∧ 可分配）
- ∃x(A(x)∨B(x)) ⇔ ∃xA(x)∨∃xB(x)  （∃ 对 ∨ 可分配）
- 注意：∀ 对 ∨、∃ 对 ∧ 的分配一般不成立！

**量词辖域收缩与扩张**（当 B 不含 x 时）：
- ∀x(A(x)∧B) ⇔ ∀xA(x)∧B
- ∀x(A(x)∨B) ⇔ ∀xA(x)∨B
- ∃x(A(x)∧B) ⇔ ∃xA(x)∧B
- ∃x(A(x)∨B) ⇔ ∃xA(x)∨B
- ∀x(A(x)→B) ⇔ ∃xA(x)→B
- ∃x(A(x)→B) ⇔ ∀xA(x)→B

**量词交换等值式**：
- ∀x∀yF(x,y) ⇔ ∀y∀xF(x,y)（同种量词可交换）
- ∃x∃yF(x,y) ⇔ ∃y∃xF(x,y)（同种量词可交换）
- 不同种量词不可随意交换！

### 前束范式

- **前束范式**：所有量词都移到公式最前面，辖域延伸到公式末尾，且不含自由变元。
  形式：Q₁x₁ Q₂x₂ ... Qₙxₙ B（B 是不含量词的母式）

**求前束范式的步骤**：
1. 消去 → 和 ↔（用蕴含等值式和等价等值式）
2. 用**换名规则**：如果两个量词的指导变元同名或辖域有冲突，改名字
3. 用**量词否定等值式**把否定推进去
4. 用**辖域扩张**规则把量词逐步外提到最前面

> **注意**：不换名同名冲突是扣分重灾区！

### PPT经典例题

**例** 求 ¬∀x(F(x,y)∧∃yG(x,y))→H(x) 的前束范式。

解：
① ¬∀x(F(x,y)∧∃yG(x,y))→H(x)
② ⇔ ¬∀x(F(x,y)∧∃zG(x,z))→H(x)  （换名：把内部的 y 换成 z）
③ ⇔ ∃x¬(F(x,y)∧∃zG(x,z))→H(x)  （量词否定）
④ ⇔ ∃x(¬F(x,y)∨¬∃zG(x,z))→H(x)  （德摩根律）
⑤ ⇔ ...

## 2.5 一阶逻辑推理理论 🔴

### 四个核心规则

在谓词逻辑的推理系统 NL 中，有四个特殊的规则：

**UI（全称实例化）**：
- ∀xA(x) ⇒ A(c)
- 含义："所有 x 满足 A" ⇒ 任取一个具体个体 c，A(c) 为真
- **无条件使用**。

**UG（全称泛化）**：
- A(c) ⇒ ∀xA(x)
- 含义：如果对**任意的** c（不附带任何特殊假设），A(c) 成立，那么对所有 x 成立
- **关键限制**：c 必须是"任意的"个体，不能来自特定前提中的假设。

**EI（存在实例化）**：
- ∃xA(x) ⇒ A(c)
- 含义：如果"存在满足 A 的 x"，给它起个名字 c
- **关键限制**：c 必须是**新引入的**常元，不能出现在前提中，也不能是之前已经出现过的。

**EG（存在泛化）**：
- A(c) ⇒ ∃xA(x)
- 含义：如果某个具体个体 c 满足 A，那么"存在满足 A 的个体"为真
- **无条件使用**。

### 四个规则使用顺序（必考）

> **最重要的原则**：**先处理存在量词（EI），再处理全称量词（UI）**。
>
> 原因：EI 引入的新常元不能出现在前提中。如果先用了 UI 把全称量词实例化成含某个常元的公式，后面用 EI 时如果恰好引入了同名常元就会冲突。

**典型证明步骤**：
1. 引入前提
2. **先**对 ∃x... 用 EI（引入新常元）
3. 然后对 ∀x... 用 UI（可以用 EI 引入的常元作为 c）
4. 用命题逻辑的推理规则推导
5. 用 EG 加回存在量词，或用 UG 加回全称量词

### EI 规则的限制条件（最容易错的地方）

EI 有两个铁律：
1. c 必须是**新常元**——前面没出现过的
2. c 不能是**自由变元**——EI 只能引入常元

> **经典错误**：前提里有 ∃xP(x) 和 ∃xQ(x)。如果对第一个用 EI 得到 P(c)，对第二个用 EI 时**不能说 Q(c)**，因为 c 已经被 P 用过了！必须用新常元 d，得到 Q(d)。P(c) 和 Q(d) 里的两个个体"不一定相同"。

## 本章脉络与对比总结

### 知识递进关系

```
个体词 + 谓词 + 量词 → 一阶逻辑公式（合式公式）
                              ↓
                        解释与赋值 → 公式类型
                              ↓
                        等值式（量词否定、辖域扩张等）
                              ↓
                        前束范式（量词全部前移）
                              ↓
                        推理理论（UI/UG/EI/EG）
                              ↓
                        自然推理系统 NL 中的证明构造
```

### 第1章 vs 第2章 对比

| 维度 | 命题逻辑（第1章） | 谓词逻辑（第2章） |
|------|-------------------|---------------------|
| 分析粒度 | 整句（命题） | 句子内部（个体+谓词+量词） |
| 表达能力 | 无法表达"所有""存在" | 可表达任意量化关系 |
| 真值决定 | 命题变元的赋值 | 解释 I + 赋值 σ |
| 永真的判断 | 真值表（有限） | 无法穷举 |
| 范式 | 主析取/主合取范式 | 前束范式 |
| 推理规则 | 假言推理、拒取式等 | 命题逻辑规则 + UI/UG/EI/EG |

### 关键桥梁

- **量词否定等值式** ¬∀x ⇔ ∃x¬ 和 ¬∃x ⇔ ∀x¬ 是谓词逻辑版的德摩根律
- **换名规则**是求前束范式的技术前提
- **UI 和 EI 的先后顺序**决定了谓词推理证明的成败

### 考试应对策略

1. **量词否定等值式**：必背。口诀："∀变∃，∃变∀，否定往右推"
2. **前束范式**：几乎必考。步骤机械——消→→换名→否定推进→量词外提
3. **UI/UG/EI/EG**：四个规则的名字、符号、条件要能默写。EI 的限制条件是高频考点
4. **谓词推理证明**：核心策略——先 EI 后 UI
5. **自由变元 vs 约束变元的判断**：拿笔画辖域，逐层判断
