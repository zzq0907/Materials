## 3.1 集合基本概念 🔴

### 什么是集合

集合由一些确定的、互不相同的对象构成。这些对象称为**元素**。

- **集合的表示**：
  - **列举法**：A = {1, 2, 3, 4}
  - **描述法**：B = {x | x 是小于 10 的质数} = {2, 3, 5, 7}

- **元素与集合的关系**：∈（属于）、∉（不属于）。
- **集合相等**：外延公理——两集合元素完全相同则相等。{1, 2} = {2, 1} = {1, 2, 2}。

### 子集与真子集

- **子集**：A ⊆ B ⇔ ∀x(x∈A → x∈B)
- **真子集**：A ⊂ B ⇔ A ⊆ B ∧ A ≠ B

> **证明 A⊆B 的标准方法**：任取 x∈A，推导出 x∈B。

### 特殊集合

- **空集 ∅**：不含任何元素的集合，唯一，是任何集合的子集
- **全集 E / U**：包含讨论范围内所有元素的集合
- **幂集**：P(A) = {x | x ⊆ A}，|P(A)| = 2^{|A|}

> **幂集大小为什么是 2^{|A|}**：A 有 n 个元素，每个元素都有"选"或"不选"两种可能，所以子集总数是 2ⁿ。

## 3.2 集合运算 🔴

### 五种基本运算

| 运算 | 记号 | 定义 | 通俗理解 |
|------|------|------|----------|
| 并 | A∪B | {x | x∈A ∨ x∈B} | 两个集合"合起来" |
| 交 | A∩B | {x | x∈A ∧ x∈B} | 两个集合"公有的" |
| 差（相对补） | A−B | {x | x∈A ∧ x∉B} | 属于 A 但不属于 B |
| 补（绝对补） | ∼A | E−A | 全集里去掉 A |
| 对称差 | A⊕B | (A−B)∪(B−A) | "只在一方"的元素 |

### 运算律

与命题逻辑的等值式几乎一一对应（∧ ↔ ∩，∨ ↔ ∪，¬ ↔ ∼）：

- **幂等律**：A∪A=A，A∩A=A
- **交换律**：A∪B=B∪A，A∩B=B∩A
- **结合律**：(A∪B)∪C=A∪(B∪C)
- **分配律**：A∪(B∩C)=(A∪B)∩(A∪C)
- **吸收律**：A∪(A∩B)=A，A∩(A∪B)=A
- **德摩根律**：∼(A∪B)=∼A∩∼B，∼(A∩B)=∼A∪∼B

> **和命题逻辑的对应关系**：这就是为什么学命题逻辑——它的代数结构在集合论里原样复用。∪ 对应 ∨，∩ 对应 ∧，∼ 对应 ¬，E 对应 1，∅ 对应 0。

## 3.3 容斥原理 🟠

### 什么是容斥原理

容斥原理解决"数并集大小"的问题。直接加会把重复的算多次，所以要"先加后减"。

**两个集合**：
$$|A ∪ B| = |A| + |B| - |A ∩ B|$$

**三个集合**：
$$|A ∪ B ∪ C| = |A| + |B| + |C| - |A∩B| - |A∩C| - |B∩C| + |A∩B∩C|$$

**一般形式（n 个集合）**：
$$\left|\bigcup_{i=1}^n A_i\right| = \sum_{i} |A_i| - \sum_{i<j} |A_i∩A_j| + \sum_{i<j<k} |A_i∩A_j∩A_k| - ... + (-1)^{n-1} |A_1∩...∩A_n|$$

> **规律**：符号交替（+ − + − ...），项数依次是 C(n,1), C(n,2), ..., C(n,n)。

### 典型应用

- **欧拉函数 φ(n)**：1 到 n 中与 n 互质的数的个数
- **错位排列**：n 个元素都不在原来位置上的排列数
- **满射计数**：A 到 B 的满射函数个数

## 3.4 有序对与笛卡尔积 🟠

### 有序对

- **有序对**：⟨a, b⟩，有顺序！⟨a, b⟩ ≠ ⟨b, a⟩（除非 a=b）。

### 笛卡尔积

- **笛卡尔积**：A × B = {⟨x, y⟩ | x∈A ∧ y∈B}
- 大小：|A × B| = |A| × |B|
- 性质：A × (B∪C) = (A×B) ∪ (A×C)（× 对 ∪ 满足分配律）
- A × B = B × A 一般不成立（除非 A=B）

## 3.5 二元关系 🔴

### 什么是关系

- **二元关系**：R ⊆ A × B，即 R 是笛卡尔积的一个子集。
- 若 R ⊆ A × A，则称 R 为 **A 上的关系**。

> **理解**：关系就是"有联系"的有序对的集合。例如 A={1,2,3}，"小于关系" R = {⟨1,2⟩, ⟨1,3⟩, ⟨2,3⟩}。

### 三种特殊关系

- **空关系**：∅，没有任何元素有关系
- **全域关系**：Eₐ = A × A，任何两个元素都有关系
- **恒等关系**：Iₐ = {⟨x, x⟩ | x∈A}，每个元素只和自己有关系

### 关系的三种表示法

1. **集合表达式**：直接列举有序对，如 R = {⟨1,2⟩, ⟨2,3⟩}
2. **关系矩阵 M_R**：mᵢⱼ=1 当且仅当 ⟨aᵢ, aⱼ⟩∈R，否则为 0
3. **关系图**：顶点代表元素，有向边代表有序对

> **三种表示各有优势**：集合表达式写起来简单，矩阵最适合计算（闭包、复合），图最直观（判断性质时一目了然）。

## 3.6 关系的运算 🔴

### 基本运算

**逆关系**：R⁻¹ = {⟨y, x⟩ | ⟨x, y⟩∈R}
- 矩阵上：M_{R⁻¹} = M_Rᵀ（转置）

**复合关系**：F∘G = {⟨x, z⟩ | ∃y(⟨x, y⟩∈F ∧ ⟨y, z⟩∈G)}
- 矩阵上：M_{F∘G} = M_F × M_G（布尔矩阵乘法，+ 换成 ∨，× 换成 ∧）

**幂运算**：R⁰ = Iₐ（恒等关系），Rⁿ⁺¹ = Rⁿ ∘ R

### 闭包

闭包 = 给关系"补上缺失的元素"，使其满足某种性质，同时尽量少加新元素。

| 闭包 | 记号 | 公式 | 含义 |
|------|------|------|------|
| **自反闭包** | r(R) | R ∪ Iₐ | 补上所有 ⟨x,x⟩ |
| **对称闭包** | s(R) | R ∪ R⁻¹ | 每条单向边补上反向边 |
| **传递闭包** | t(R) | ⋃ᵢ₌₁^∞ Rⁱ | 每条路径补上直达边 |

**用矩阵求闭包**：
- M_{r(R)} = M_R ∨ I（主对角线全改成 1）
- M_{s(R)} = M_R ∨ M_Rᵀ（矩阵和转置取或）
- M_{t(R)} = M_R ∨ M_R² ∨ ... ∨ M_Rⁿ（n 次幂取或）

## 3.7 关系的性质 🔴

### 五大性质一览

判断关系的性质是本章最核心的技能。三种角度看：**定义、矩阵、关系图**。

| 性质 | 定义 | 矩阵特征 | 图特征 |
|------|------|----------|--------|
| **自反性** | ∀x∈A, ⟨x,x⟩∈R | 主对角线全 1 | 每个顶点有环 |
| **反自反性** | ∀x∈A, ⟨x,x⟩∉R | 主对角线全 0 | 每个顶点无环 |
| **对称性** | ⟨x,y⟩∈R→⟨y,x⟩∈R | 对称矩阵 | 有边必有反向边 |
| **反对称性** | ⟨x,y⟩∈R∧⟨y,x⟩∈R→x=y | rᵢⱼ∧rⱼᵢ=0 (i≠j) | 无双向边（除环外） |
| **传递性** | ⟨x,y⟩∈R∧⟨y,z⟩∈R→⟨x,z⟩∈R | M_R² 中 1 的位置 M_R 中也为 1 | 有路径则有直接边 |

> **自反≠反自反，两者互斥？不！** 一个关系可以既不自反也不反自反。比如矩阵对角线上有 1 有 0。

> **最容易出错的**：传递性！必须检查每一个"两步路径"是否都有"一步直达边"。反例：R={⟨1,2⟩, ⟨2,3⟩}，没有 ⟨1,3⟩，不传递。

### 判断性质的步骤

1. **看图最快**：画关系图，直观判断
2. **看矩阵**：检查对角线、对称性、M_R² ≤ M_R（传递性）
3. **用定义**：严格证明时必须用定义

## 3.8 等价关系与划分 🔴

### 等价关系

- **等价关系**：同时满足**自反、对称、传递**的关系。记作 ∼。
- 典型例子："等于"、"同班同学"、"模 n 同余"。

> **记忆**：等价关系 = "三个都满足"（自反 + 对称 + 传递）。偏序关系 = 自反 + 反对称 + 传递。两者差在对称 vs 反对称。

### 等价类与商集

- **等价类**：[x]_R = {y | y∈A ∧ ⟨x, y⟩∈R}
- **商集**：A/R = {[x]_R | x∈A}
- 同一等价类里任意两个元素互相等价
- 不同等价类不相交
- 所有等价类的并 = A

### 划分

- **划分**：集合 A 的一族非空子集，两两不交且并为 A。

**核心定理**：等价关系与划分**一一对应**。
- 给定等价关系 → 它的等价类构成一个划分
- 给定一个划分 → 定义"属于同一块"的关系就是一个等价关系

> **这一定理的意义**：等价关系和划分是"同一个硬币的两面"。考试中经常要求"由划分构造等价关系"或反过来。

## 3.9 偏序关系 🔴

### 偏序关系

- **偏序关系**：同时满足**自反、反对称、传递**的关系，记作 ≼。
- **偏序集（poset）**：⟨A, ≼⟩
- 典型例子：整数的 ≤、集合的 ⊆、整除关系。

> **为什么叫"偏"序**：因为不是任意两个元素都可比较。比如集合 {1,2} 和 {2,3}，谁都不是对方的子集——它们不可比。

### 可比、覆盖与哈斯图

- **可比**：x ≼ y 或 y ≼ x，两者至少一个成立。
- **覆盖**：y 覆盖 x，如果 x < y 且中间没有其他元素。
- **哈斯图**：用覆盖关系简化的有向图——只画"最直接的上下级关系"。

**画哈斯图的规则**：
1. 省去所有环（自反性隐含）
2. 省去所有可由传递性推出的边
3. 位置低的元素放下面、高的放上面（从下往上，不画箭头）

### 特殊元素

| 概念 | 定义 | 唯一性 |
|------|------|--------|
| **最大元** | 比所有元素都大 | 若存在，唯一 |
| **最小元** | 比所有元素都小 | 若存在，唯一 |
| **极大元** | 没有比它更大的 | 不一定唯一 |
| **极小元** | 没有比它更小的 | 不一定唯一 |
| **上界** | 比子集中所有元素都大 | 不一定唯一 |
| **下界** | 比子集中所有元素都小 | 不一定唯一 |
| **上确界** | 最小的上界 | 若存在，唯一 |
| **下确界** | 最大的下界 | 若存在，唯一 |

> **区分极大元和最大元**：最大元比所有人（包括自己）都大；极大元只是"没人比我大"。哈斯图顶上可能有好几个极大元（互相不可比）。

### 全序与良序

- **全序（线序）**：任意两元素可比。如实数的大小关系。
- **良序**：全序 + 每个非空子集都有最小元。如自然数。

## 3.10 函数 🔴

### 函数的基本概念

- **函数（映射）**：f: A → B，∀x∈A 存在**唯一**的 y∈B 使 ⟨x, y⟩∈f。
- 函数是**特殊的关系**：每个 x 恰好对应一个 y。

> **函数 vs 关系**：函数一定是关系，但关系不一定是函数。关系允许一个 x 对应多个 y，函数不允许。

### 函数的三类性质

| 性质 | 定义 | 通俗理解 |
|------|------|----------|
| **满射 (onto)** | ∀y∈B, ∃x∈A, f(x)=y | B 中每个元素都被"射中" |
| **单射 (1-1)** | f(x₁)=f(x₂) → x₁=x₂ | 不同输入对应不同输出 |
| **双射 (bijection)** | 既单又满 | 一一对应 |

> **有限集上的函数计数**：
> - A→B 的函数总数：|B|^{|A|}
> - A→B 的单射个数（需 |A|≤|B|）：P(|B|, |A|)
> - A→B 的双射个数（需 |A|=|B|）：|A|!

### 复合与反函数

- **函数复合**：(g∘f)(x) = g(f(x))，先 f 后 g
- **反函数 f⁻¹**：当且仅当 f 是**双射**时存在

## 本章脉络与对比总结

### 知识递进关系

```
集合基本概念 → 集合运算 → 容斥原理（计数应用）
                ↓
          有序对 → 笛卡尔积 → 二元关系
                                ↓
                    关系的运算（逆、复合、闭包）
                                ↓
                    关系的性质（五性判断）
                           ↙        ↘
                    等价关系          偏序关系
                    （自反对称传递）   （自反反对称传递）
                        ↓                ↓
                    划分与商集        哈斯图与特殊元素
                        ↓
                    函数（特殊的关系）
                        ↓
                    单射/满射/双射
```

### 三大"特殊关系"对比

| 性质 | 等价关系 | 偏序关系 | 函数 |
|------|----------|----------|------|
| 自反 | ✅ | ✅ | 不要求 |
| 对称 | ✅ | ❌（反对称） | 不要求 |
| 传递 | ✅ | ✅ | 不要求 |
| 唯一性 | 无 | 无 | **每个 x 对应唯一 y** |
| 典型例子 | 等于、同余 | ≤、⊆、整除 | f(x)=x² |

### 关系性质的矩阵判据速查

| 性质 | 矩阵判据 |
|------|----------|
| 自反 | 对角线全 1 |
| 反自反 | 对角线全 0 |
| 对称 | 矩阵 = 转置 |
| 反对称 | 对称位置不同时为 1（对角线除外） |
| 传递 | M_R² ≤ M_R（布尔意义下） |

### 考试应对策略

1. **关系性质的判断**：几乎必考一道。画关系图是最快的方法，但证明必须用定义
2. **哈斯图**：必有一题。记住：去掉环+去掉传递边+下小上大
3. **等价关系与划分**：概念互推是常考题型
4. **函数类型判定**：有限集上多用计数公式，无限集上用定义证明
5. **容斥原理**：符号交替规律，从 1 个集合加到 n 个集合

### 【2019-2020 Q1】容斥原理（5分）

> 某班共有30名学生，每名同学均至少选择了选修课I、II、III中的一门。已知选择了选修课I的同学有20名，选择了选修课II的同学有15名，选择了选修课III的同学有10名；同时选择了选修课I和II的同学有7名，同时选择了选修课II和III的同学有5名，同时选择了选修课I和III的同学有5名，试求同时选择了三门选修课的同学人数。

**知识点链接**：→ [[#3.3 容斥原理]]
**解法**：|I∪II∪III| = |I|+|II|+|III| − |I∩II|−|I∩III|−|II∩III| + |I∩II∩III|，即 30 = 20+15+10−7−5−5+x，x = 2。
